# 💰 Personal Expense Tracker (C++)

### 📝 Project Description
A simple C++ application to track daily expenses with data persistence using file handling. It saves your data to a text file named `expenses.txt`.

### 🚀 Features
* **Add Expense:** Record description and amount.
* **Data Persistence:** Saves data even after closing the program.
* **Total Calculation:** Shows the sum of all expenses.
* **Clear Records:** Option to reset the data.
